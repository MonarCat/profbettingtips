# profbettingtips
Daily football predictions, free match insights, paid premium odds.

## Deploying on Vercel

This repository is initialized for static deployment on Vercel.

- Entry page: `index.html`
- Daily page: `daily.html`
- Styles: `css/style.css`
- Scripts: `js/app.js`
- Vercel config: `vercel.json`
